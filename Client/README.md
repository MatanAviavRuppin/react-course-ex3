# react-course-ex3

## Requirements:
- NodeJS: at least version 18.12.1
- NPM: at least version 8.19.2

<br />

## Instructions:
- Go to: ```/src/classes/WebSettings.js```
and change ```url``` static property to the correct API endpoint
- At the root of the project folder, run ```npm install```
- run ```npm start```
